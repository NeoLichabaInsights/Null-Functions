# Null-Functions

This project contains simple SQL exercises focused on creating tables, inserting data, and handling missing values.

What this project covers:

The examples demonstrate how to:

• Create tables with different data types
• Insert values, including NULL
• Use IFNULL() IS NULL, COALESCE and CASE statements
• Replace missing values with clear default text
• Convert dates and numbers into strings for display
• Keep the original column data type unchanged
• Correct common SQL syntax errors

# Purpose:

To practise writing clear SQL queries and working confidently with missing data in Databricks.
